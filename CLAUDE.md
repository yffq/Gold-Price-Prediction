# Gold Price Prediction

基于宏观经济因素的每日金价预测项目。

## 项目结构

```
predict/                    — 每日预测报告 (markdown)
.claude/skills/             — 分析框架 (gold-price-analysis)
.claude/commands/           — 自定义命令 (/predict)
local_media/                — 本地媒体资料 (不上传 GitHub)
```

## 使用方式

在本项目目录下运行 Claude Code，输入 `/predict` 即可生成当日预测报告。
