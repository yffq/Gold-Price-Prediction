# /predict - 生成当日黄金价格预测报告

使用 `gold-price-analysis` skill 框架生成预测报告并保存到 `predict/` 目录。

## 执行步骤

1. **获取最新市场数据**（使用网络搜索）：
   - 国际金价 (美元/盎司)
   - 上海金交所 Au9999 (元/克)
   - 美元指数 DXY
   - SPDR GLD 持仓 (吨)
   - 美联储最新利率和政策动向
   - 本周重要经济数据日程

2. **基于 4 步框架分析**：
   - **实际利率趋势**：联邦基金利率 - 核心 PCE，下降看多，上升看空
   - **央行购金行为**：查世界黄金协会数据，净买入看多
   - **ETF 资金流向**：查 GLD 持仓变化，增持看多，价涨仓平则警惕
   - **交易时段特征**：美盘主导上涨=机构认可，亚盘主导=散户 FOMO

3. **生成报告**，格式如下：

```markdown
# 黄金价格预测报告

**预测日期**: {YYYY-MM-DD}
**预测周期**: 7天

## 一、当前市场数据
（用表格列出 5 个指标的数值和变化）

## 二、4步框架分析
### 1. 实际利率趋势
### 2. 央行购金行为
### 3. ETF资金流向
### 4. 交易时段特征

## 三、7日价格预测
（用表格列出 7 天的：日期、美元区间、人民币区间、方向、置信度）

## 四、本周关键事件

## 五、风险提示
（上行和下行风险各 2-3 条）

## 六、总结
**7日总体方向**: 看涨/看跌/震荡
**核心逻辑**: 一句话
**操作建议**: 具体建议

---
*AI预测报告，仅供参考，不构成投资建议*
```

4. **保存报告**到 `predict/{YYYY-MM-DD}_gold_prediction.md`
